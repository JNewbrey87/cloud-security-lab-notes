# Cloud Security Lab Notes

## Overview

This repository is a working archive of hands-on lab output, study notes, and
reusable artifacts produced while building cloud security depth in Microsoft Azure.
Content is added continuously as cert work and practical experience accumulates.

## Contents

### KQL Queries
Kusto Query Language queries written for Microsoft Sentinel, covering threat detection,
log analysis, and security monitoring use cases.

### Sentinel Analytic Rules
Custom detection rules and alert configurations developed during SC-200 study and lab work.

### Azure Policy Definitions
Policy-as-code artifacts for governance, compliance enforcement, and configuration
management in Azure environments.

### Lab Notes
Structured notes from SC-200, AZ-500, and CySA+ study covering key concepts,
gotchas, and practical application.
